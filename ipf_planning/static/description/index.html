Description

This Process-API integration contains information for Mina Sidor and ALF needs. Specifically for Odoo it returns whether a jobseeker's planning information is in Planeringsverktyget (PLV), BÄR or absent. More detailed flow documentation is available at the IPF API Portal: Arbetssokande Planering, endpoint /no-jwt/handlingsplaner.

This integration requires basic authentication. Contact integration team for more information.

Configuration

To configure this integration, you need to be logged in as Super User:

  1. Go to Tekniskt > IPF Settings
  2. Select IPF Planning
  3. Enter the following parameters
    • Url: https://ipf-acc.arbetsformedlingen.se
    • Port: 443
    • Client ID: is available in the IPF Portal
    • Client Secret: is available in the IPF Portal
    • Auth User: AfCrm
    • Auth Password: is available from integration team
    • AF-Environment: T2
    • AF-Systemid: AFCRM

Test

To test this integration, you need to:

  1. Go to Tekniskt > Åtgärder >Serveråtgärder
  2. Select IPF Planning
  3. The env['af.ipf'].browse() should be set to IPF Planning's current enviroment ID. i.e 8
  4. The personnummer should be set to an jobseeker. i.e 196904180012
  5. Run should return a list with the jobseeker's plan
  6. Return to Jobseeker tab and confirm .
  7. Go to Arbestsökande > Kundens planering > klick Visa handlingsplan
  8. See example below
