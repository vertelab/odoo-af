Calendar management - AF

This module manage and schedule meetings between jobseekers and employees of Arbetsförmedlingen.

Configuration

To configure this module, you need to:

  1. Go to Calendar > employee > local to see the list
  2. Observe dates and times.
  3. To be updated.

Usage
