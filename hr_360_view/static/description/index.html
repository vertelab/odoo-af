Modul formulär Arbetssökande: handläggares “Arbetsyta”



Den här modulen lägger till huvudsida för vyn ”Arbetsyta”,
där kan man ”söka arbetssökande” via flikarna ”Vid kontakt” och
”Utifrån myndighetens behov” med uppgifter centrerade från handläggaren.
Man måste alltid börja med att ange ”sökorsak” och där får man olika
alternativ till sökningen som man kan välja mellan,
sedan kan man välja att specificera sig med antingen personnumret,
kundnumret eller e-postadressen.



Jiror som ligger till grund för denna modul:



- v12.0.0.1 AFC-667 - HR 360 View Module.
Lägger till huvudsida för handläggare med sökfunktion med uppgifter centrerade från handläggaren.
- v12.0.0.2 AFC-713
Bytt namn från Handläggaryta till Arbetsyta enligt beslut från Införandegruppen. Dolt menyn HR 360.
- v12.0.0.3 AFC-715
Lagt till rubrik på vyn för Arbetsyta.
- v12.0.0.4 AFC-715
Lade till beroende på Firstname Extension som döljer förnamn-efternamn-fälten i visningsläge.
