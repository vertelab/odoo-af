Modul formulär Arbetssökande: handläggares “Arbetsyta”



Den här modulen lägger till huvudsida för vyn ”Arbetsyta”,
där kan man ”söka arbetssökande” via flikarna ”Vid kontakt” och
”Utifrån myndighetens behov” med uppgifter centrerade från handläggaren.
Man måste alltid börja med att ange ”sökorsak” och där får man olika
alternativ till sökningen som man kan välja mellan,
sedan kan man välja att specificera sig med antingen personnumret,
kundnumret eller e-postadressen.
